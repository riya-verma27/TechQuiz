This Quiz App is designed in Android Studio using JAVA,XML and FireBase.It consists of the questions of different categories such as GK,Programming,Maths ,Operating System and DBMS.Each category consists of different number of sets.Each set consists of different number of questions related to that category.For each question,there is a timer of 10 seconds.After the completion of all questions, score will be displayed.
